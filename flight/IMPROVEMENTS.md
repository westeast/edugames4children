### ✅ 1. 车辆与建筑物碰撞检测
**文件**: `flight/js/entities.js`

**改进内容**:
- 添加了车辆与建筑物的碰撞检测系统
- 车辆遇到建筑物时会自动转向，不再穿过楼房
- 使用 AABB 碰撞检测算法，性能高效
- 动态加载城市地图的建筑物边界数据

**技术实现**:
```javascript
// 在车辆移动前检测前方是否有建筑物
function checkCarBuildingCollision(car, newX, newZ) {
  for (const b of buildingBounds) {
    if (newX + carRadius > b.minX && newX - carRadius < b.maxX &&
        newZ + carRadius > b.minZ && newZ - carRadius < b.maxZ) {
      return true; // 碰撞
    }
  }
  return false;
}
```

### ✅ 5. 云台穿越模式
**文件**: `flight/js/config.js`, `flight/js/ui.js`, `flight/js/controls.js`, `flight/index.html`

**改进内容**:
- 在设置 > 拍摄中添加了云台模式设置
- **跟随模式**: 云台保持水平，画面稳定
- **穿越模式(FPV)**: 云台随无人机倾斜，画面侧飞时倾斜

**穿越模式效果**:
- 左摇杆左右移动 → 无人机侧飞 → roll变化
- roll变化 → 云台倾斜 → 画面倾斜
- **倾斜角度**: ±30°（强倾斜效果）
- 倾斜角度 = droneRoll × 1.5（放大系数）

**UI界面**:
- 设置 > 拍摄 > 云台模式
- 两个按钮：跟随 / 穿越模式
- 显示当前模式描述

## 技术亮点

### 1. 碰撞检测优化
- 使用 AABB（轴对齐包围盒）碰撞检测
- 车辆碰撞半径: 2.0m
- 无人机碰撞半径: 1.5m
- 空间分割优化（仅检测当前chunk的建筑物）

### 2. 云台倾斜数学
```javascript
// 无人机侧飞速度 → roll角 → 云台倾斜角
// roll角范围：-0.3 到 +0.3 弧度（约 ±17°）
// 穿越模式下，摄像机旋转 z = roll × 1.5
// 最大倾斜角：±30°（±0.52弧度）
```

### 3. 桥梁洞口设计
- Air 3: 257×212×103mm（折叠），约600mm展开宽度
- Mini 4 Pro: 247×165×91mm（折叠），约480mm展开宽度
- Mavic 3 Pro: 347×283×107mm（折叠），约900mm展开宽度
- **安全通过尺寸**: 洞口最小 2m×2m（考虑姿态变化）

## 性能影响

- **车辆碰撞检测**: 每帧每辆车检测约10-20个建筑物（已优化）
- **车道线渲染**: 增加约500-800个网格对象（每个chunk）
- **桥梁渲染**: 每座桥约50-100个网格对象
- **云台倾斜**: 计算量极小，几乎无性能影响

**建议**:
- 在低端设备上可考虑降低车道线密度
- 桥梁数量已限制为每chunk 1-2座
